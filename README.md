This project is set up to be run in IntelliJ IDEA.

1.  Clone the repository.
2.  Open the project in IntelliJ IDEA.
3.  To run the solution for a specific day, open the corresponding `.kt` file (e.g., `src/day1/Day1_1and2.kt`) and run the `main()` function inside it.
